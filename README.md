## 📄 Description
I have designed an age calculator using HTML, CSS, and JS. This project is a simple web application that takes a user's birthdate as input and calculates their age.

## 🎯 Features
* Calculates age based on birthdate
* Uses HTML, CSS, and JS for a dynamic user interface
* Simple and easy to use

## 🌐 Tech Stack
* HTML: Used for structuring the content
* CSS: Used for styling the application
* JS: Used for adding interactivity to the application

## 📁 Project Structure
* `README.md`: This file, containing information about the project
* `index.html`: The main entry point of the application
* `script.js`: Contains the JavaScript code for the application
* `style.css`: Contains the CSS styles for the application

## 📦 Installation
1. Clone the repository using `git clone https://github.com/kiranShamsHere/age_calculator.git`
2. Open the `index.html` file in a web browser to use the application

## 🚀 Usage
1. Open the `index.html` file in a web browser
2. Enter your birthdate in the input field
3. Click the "Calculate Age" button to see your age
